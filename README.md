# CLI Text Adventure Game

A command-line based interactive story game built using Python.

## 🎮 Features

- Choice-based gameplay
- Multiple story paths and endings
- Replay system
- Input validation
- Clean CLI interface

## 🧠 How it works

The game uses a dictionary-based story structure where:
- Each scene contains text, choices, and next paths
- Player choices determine the story flow
- Some paths lead to different endings

## ▶️ How to run

1. Make sure Python is installed
2. Install required packages (if needed)
3. Run the script:

python main_script.py
